# Sacred Core - Hybrid Pricing Model

## Overview

**Hybrid subscription + usage-based credits system**:
- 3 subscription tiers (Starter, Pro, Enterprise)
- Monthly included credits per tier
- Pay-as-you-go overage: $0.01 per credit
- Self-serve credit packs ($4.99–$89.99)
- 10% annual discount on subscriptions
- Free first campaign for all new users

---

## Subscription Features

### Monthly Billing
- Charged on same day each month
- Credit balance resets on configurable day (default: 1st)
- Unused credits from previous month carry over or reset (configurable)
- Overage billed at $0.01/credit

### Annual Billing
- **10% discount**: Annual price = (Monthly price × 12) × 0.9
- **Pro Annual**: $490 (vs $588 monthly)
- **Enterprise Annual**: $1,990 (vs $2,388 monthly)
- **Savings**: $98/year (Pro), $398/year (Enterprise)
- Renews automatically, cancellable anytime

### Upgrades & Downgrades
- **Upgrade**: Prorated charge for difference (e.g., Starter → Pro mid-month)
- **Downgrade**: Prorated credit refund (e.g., Pro → Starter, lose 1.5k credits)
- **Cancel**: Immediate, remaining credits stay until end of billing period

**Rationale**: Removes friction, proves value, drives conversion to paid.

## Stripe Integration

### Webhooks (Firebase Functions)

```
POST /webhooks/stripe
├─ checkout.session.completed
│  ├─ subscription created → Add credits to user
│  └─ credit pack purchased → Add credits to user
│
├─ customer.subscription.updated
│  ├─ Tier changed → Update user profile
│  └─ Plan changed → Prorate credits/refunds
│
├─ customer.subscription.deleted
│  ├─ Downgrade user to starter
│  └─ Keep remaining credits
│
├─ invoice.payment_succeeded
│  ├─ Monthly subscription renewed → Reset credits
│  └─ Log transaction
│
└─ invoice.payment_failed
   ├─ Retry 3 times
   ├─ After 3 failures, pause features
   └─ Send user notification
```

## Refund & Dispute Policy

### Subscription Cancellation
- **Immediate**: Lose access to tier features
- **Credits**: Remaining credits = user keeps them until expired
- **Refund**: None (subscription is recurring until canceled)
- **Dispute**: 30-day money-back guarantee via Stripe Disputes

### Credit Pack Refunds
- **Within 7 days**: Full refund if no credits used
- **Partial use**: Prorated refund (e.g., used 500 of 1000 credits = 50% refund)
- **After 7 days**: No refunds (credits are consumable)

### Overage Chargesback
- **Dispute**: Automatic retry 3 times
- **Failed payment**: Feature lock until resolved
- **Goodwill**: 1-time reversal per customer (support discretion)
