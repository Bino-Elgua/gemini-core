## QUICK COMMANDS

### Start Development
```bash
cd sacred-core
npm install
npm run dev
# Opens http://localhost:1111
```

### Run Tests
```bash
# Full E2E test suite
npx playwright test tests/e2e/comprehensive.spec.ts

# Show browser during tests
npx playwright test --headed

# Debug mode
npx playwright test --debug
```

### Build & Deploy
```bash
# Production build
npm run build

# Deploy to staging
npm run deploy:staging

# Run performance checks
npm run perf:check

# Security audit
npm run security:audit
```
